## Without Docker

### Package requirements for your machine:

- node v8.x.x or v9.x.x
- npm v6.x.x or newer
- Yarn v1.3 or newer

### Setup

```bash
# Clone the repository
$ git clone https://github.com/mehtamanan/dao-core.git

# Install local node dependencies:
$ npm i

# To run
$ npm run dev
```
